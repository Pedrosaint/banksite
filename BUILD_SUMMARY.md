## 🔧 Customization Examples

### Change Brand Colors

Edit `tailwind.config.js`:

```javascript
colors: {
  primary: '#2563eb',  // Your color
  secondary: '#10b981',
  accent: '#f59e0b',
}
```

### Update Content

Edit individual components in `src/components/`:

- Update text, rates, benefits
- Add more accounts/loans
- Change contact information

### Modify Styling

- Use Tailwind classes for quick styling
- Add custom CSS in component files
- Extend theme in `tailwind.config.js`

### Add New Section

1. Create `src/components/NewSection.tsx`
2. Import in `App.tsx`
3. Add section ID for scroll
4. Add to navbar navigation
